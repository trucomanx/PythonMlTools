# PythonMlTools
Python Machine Learning Tools

# Packaging

Download the source code

    git clone https://github.com/trucomanx/PythonMlTools.git

The next command generates the `dist/PythonMlTools-VERSION.tar.gz` file.

    cd PythonMlTools/src
    python setup.py sdist

For more informations use `python setup.py --help-commands`

# Install 

Install the packaged library

    pip install dist/PythonMlTools-VERSION.tar.gz
